| Attributes                                                        | Description                                                                                                                                                                                                                                        |
| ----------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **lago_id** `String` Not null                                     | Unique identifer of the credit note in Lago application                                                                                                                                                                                            |
| **sequential_id** `Integer` Not null                              | Credit note sequential id scoped on the invoice                                                                                                                                                                                                    |
| **number** `String` Not null                                      | Credit note unique number                                                                                                                                                                                                                          |
| **lago_id** `String` Not null                                     | Unique identifer of the invoice in Lago application                                                                                                                                                                                                |
| **invoice_number** `String` Not null                              | Invoice unique number                                                                                                                                                                                                                              |
| **issuing_date** `String` Not null<br /> <sub>ISO 8601 date</sub> | Date of creation of the credit note                                                                                                                                                                                                                |
| **reason** `String` Not null                                      | Issuing reason of the credit note **Possible values**\-duplicated_charge\-product_unsatisfactory\-order_change\-order_cancellation\-fraudulent_charge\-other                                                                                       |
| **credit_status** `String`                                        | Status of the credit part of the credit note **Possible values**\- available: An amount remain available for future usage.\- consumed: The amount is fully consumed. Remaining amount is 0\- voided: The remaining amount cannot be used any more. |
| **refund_status** `String`                                        | Status of the refund part of the credit note **Possible values**\- pending: The refund is pending for its execution.\- succeeded: The refund has been executed\- failed: The refund failed to execute                                              |
| **total_amount_cents** `Integer` Not null                         | Total amount in cents                                                                                                                                                                                                                              |
| **total_amount_currency** `String` Not null                       | Currency of the total amount                                                                                                                                                                                                                       |
| **credit_amount_cents** `Integer` Not null                        | Credit amount in cents                                                                                                                                                                                                                             |
| **credit_amount_currency** `String` Not null                      | Currency of the credit amount                                                                                                                                                                                                                      |
| **refund_amount_cents** `Integer` Not null                        | Refund amount in cents                                                                                                                                                                                                                             |
| **refund_amount_currency** `String` Not null                      | Currency of the refund amount                                                                                                                                                                                                                      |
| **balance_amount_cents** `Integer` Not null                       | Remaining credit amount in cents                                                                                                                                                                                                                   |
| **balance_amount_currency** `String` Not null                     | Currency of the balance amount                                                                                                                                                                                                                     |
| **vat_amount_cents** `Integer` Not null                           | VAT amount in cents                                                                                                                                                                                                                                |
| **vat_amount_currency** `String` Not null                         | Currency of the VAT amount                                                                                                                                                                                                                         |
| **sub_total_vat_excluded_amount_cents** `Integer` Not null        | Subtotal excluding tax, in cents                                                                                                                                                                                                                   |
| **sub_total_vat_excluded_amount_currency** `String` Not null      | Currency of the subtotal amount                                                                                                                                                                                                                    |
| **created_at** `String` Not null*ISO 8601 datetime*               | Date and time of creation of the credit note in UTC                                                                                                                                                                                                |
| **updated_at** `String` Not null*ISO 8601 datetime*               | Date and time of last update of the credit note in UTC                                                                                                                                                                                             |
| **file_url** `String`                                             | File url of the credit note PDF                                                                                                                                                                                                                    |
| **items** `JSON`                                                  | Array of [item](#credit-note-object) object                                                                                                                                                                                                        |

## Item object[](#item-object "Direct link to heading")

| Attributes                            | Description                                                  |
| ------------------------------------- | ------------------------------------------------------------ |
| **lago_id** `String` Not null         | Unique identifer of the credit note item in Lago application |
| **amount_cents** `Integer` Not null   | Amount in cents                                              |
| **amount_currency** `String` Not null | Currency of the amount                                       |
| **fee** `JSON` Not null               | [Fee](/docs/api/invoices/invoice-object#fee-object) object   |
